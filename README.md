## 简介

关于提高工作效率这个话题，内涵很广，此处不展开讨论。此处只是探讨和分享，如何用各种工具和方法去提高工作的效率，减少重复劳动。尤其是将一些手工处理的事情变成工具或脚本去自动处理，从而达到提高效率的目的。
